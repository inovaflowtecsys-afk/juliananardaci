# Security Policy

## Supported Versions

As atualizacoes de seguranca sao aplicadas na branch `main`.

## Reporting a Vulnerability

Nao abra vulnerabilidades em issues publicas.

Para reportar:

1. Abra uma Discussion privada com os mantenedores (se habilitado), ou
2. Envie os detalhes por canal privado definido pela equipe.

Inclua no reporte:

- Descricao tecnica do problema
- Impacto estimado
- Passos para reproduzir
- Possivel mitigacao

A equipe tentara responder em ate 3 dias uteis.
