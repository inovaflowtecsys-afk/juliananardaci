# Code of Conduct

## Our Commitment

Este projeto busca um ambiente respeitoso e colaborativo para todas as pessoas.

## Expected Behavior

- Seja respeitoso e objetivo
- Aceite feedback tecnico com profissionalismo
- Evite ataques pessoais
- Foque no problema, nao na pessoa

## Unacceptable Behavior

- Assedio, ameacas ou discriminacao
- Linguagem ofensiva
- Exposicao de dados privados sem consentimento
- Conduta que comprometa a seguranca da comunidade

## Enforcement

Mantenedores podem editar, ocultar ou remover conteudo e, em casos graves, restringir participacao.

## Scope

Aplica-se a issues, pull requests, discussions e demais canais oficiais do projeto.
